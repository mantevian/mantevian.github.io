0.4-a1 01/11/2020

- the first physics simulation test
- removed an extra console.log() in terrain generator


0.3 01/09/2020

- added squid
- added octaves in terrain gen. each octave adds some more small detail. previously this was hardcoded to be 4 but now can be set to anything from 1 to 12. more octaves = higher loading time
- all values in terrain gen are auto prefilled but if one of the fields happens to be empty the generator will use the default setting for it without crashing
- changelog now goes from new to old instead of old to new otherwise you'd have to scroll down to hell to find the newest update
- changed text colors a bit


0.2 28/08/2020

- moved a lot of files
- unified css for all pages, moved into /style.css
- redesigned the main page and this changelog page
- terraingen now has default values put in the textboxes so you don't have to type them in every time
- added these cool text boxes

- separated the biomes: now in /resources/biomes.js

- fixed a lot of biomes just never appearing
